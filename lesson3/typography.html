Typography

For this site, I went with two complementary san-serif fonts since they are easier to read on a screen. The first font is "Mandali" (with backups of Helvetica then sans-serif) which will be used for all site headings. This font provides a fresh look while still remaining professional and easy to read. The second font is "DM Sans" (with backups of Arial then sans-serif) which will be used for all site paragraphs and main content. We are using this because it is a sharper and clearer font that is easy to read, even at small sizes.

For reference, these fonts and styles are implemented throughout this site how they would be on the completed Sunny Day News site.
